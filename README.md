### Description
* You control snake with arrow keys
  **(up, down, left, right)**
* When you eat red fruit you will get
  **+1 score**
* Goal is to get the **highest score**
* When you die you can start new game 
by typing **"yes"** in given prompt, if you 
  anything else game will exit
